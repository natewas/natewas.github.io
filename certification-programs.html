Challenge

Launching a global certification program for Admins and Developers meant scaling a fair, secure, and rigorous exam process. We needed to evaluate hundreds of learners with varied workflows, prevent question fatigue, and eliminate manual processes like account creation and badge delivery.

Solution

Exam Design

  • Authored 800+ questions targeting core competencies across user types.
  • Refined questions with stakeholders to balance clarity and challenge.
  • Used pilot data to optimize for accuracy and learner experience.

Automation Workflows

  • Used Python and JavaScript to build workflows for enrollment, grading, sandbox setup, and messaging.
  • Integrated with Credly API to auto-issue digital badges upon exam completion.

Data & Feedback Loop

  • Built Domo dashboard to connect data from Qualtrics, Skilljar, and Credly.
  • Enabled real-time feedback and program health monitoring for leadership.

My Role

  • Designed and led the entire digital credential program end to end.
  • Wrote 800+ certification questions and built automation tools from scratch.
  • Ran stakeholder pilots and usability tests to refine systems and UX.
  • Built dashboards and data tools for performance tracking.

Tools & Technologies

  • Development: Python, JavaScript
  • Data & Dashboards: Domo, Qualtrics, Skilljar
  • Credentialing: Credly API
  • Communication: Dynamic email system

Impact

  • 75% decrease in admin overhead
  • Over 650 credentials issued to date
  • Scalable, repeatable model for future certifications
