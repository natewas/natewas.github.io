LMS Implementation

TL;DR: Built a learning system for 300,000+ users. Automated 1,000+ hours of manual tasks, wrote global theme code, and remapped 15,000 links. Basically, I turned a logistical nightmare into a scalable, user-friendly dream.

I led the implementation and launch of a new learning management system (LMS) designed to provide an invaluable educational platform for over 300,000 sysadmins, end users, partners, and employees. The project involved remapping and updating hyperlinks across 1,000+ courses to ensure seamless access to resources. To streamline this process, I developed automation tools using Python and Selenium, saving more than 1000 hours and significant effort during the transition.

In addition to managing content migration, I developed and maintained over 50 web pages and wrote the global theme code used across the domain, creating a cohesive and user-friendly experience. I also designed wireframes in Adobe XD to incorporate stakeholder feedback and conducted pilot testing to ensure the platform met user needs. To enhance functionality, I built front- and back-end solutions using HTML, CSS, JavaScript, and Node.js, enabling users to quickly find and navigate relevant training resources.

This project showcased my ability to deliver impactful educational tools, automate complex workflows, and design scalable, user-centric solutions that deliver exceptional value to customers.
