Localization Program

TL;DR: I designed and built an automated localization system to make MasterControl Academy course content available in 15 languages...

Challenge Solution My Role Tools Impact

Challenge

MasterControl Academy offers 750+ English-language eLearning courses...

  • Video captions remaining in English.
  • Internal navigation and links redirecting to untranslated content.
  • Poor searchability for non-English terms.

Solution

I developed an automated localization pipeline...

Custom Utility Development

  • Extracts all course text—including .vtt captions—from SCORM packages.
  • Converts content into structured JSON for translation.
  • Uses SmartCat’s CLI...
  • Reintegrates translated assets...

Full Ecosystem Translation

The system localizes not just core course content, but also:

  • Catalog pages and metadata.
  • Learning paths and UI navigation.
  • Multimedia captions and content.

Quality Control Loop

A feedback process with SMEs ensures high-quality translations...

My Role

  • Owned end-to-end design, development, and deployment.
  • Created the automation workflow...

Tools & Technologies

  • Development: Python, SmartCat CLI, JSON...
  • Authoring: Articulate Rise
  • LMS: Skilljar
  • Versioning: Git

Impact

  • 96 courses localized, scaling to 750+.
  • Up to 80% time reduction per course.
  • Automated translation of all elements.
